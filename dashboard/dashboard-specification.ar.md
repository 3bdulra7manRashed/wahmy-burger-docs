# مواصفات لوحة التحكم

---

## ١. نظرة عامة

لوحة التحكم هي نظام تشغيلي لإدارة طلبات الاستلام (Pickup) الأونلاين فقط.  
وليست بديلاً عن نظام نقاط البيع (Foodics).

### الأهداف

<ul dir="rtl">
<li>متابعة الطلبات الأونلاين</li>
<li>التحكم في تدفق الطلبات ومنع الزحام</li>
<li>إدارة أوقات الاستلام (Pickup Slots)</li>
<li>مراقبة التكامل مع Foodics</li>
<li>توفير رؤية تشغيلية واضحة</li>
</ul>

---

## ٢. المستخدمون والصلاحيات

### Super Admin

<strong>الوصف:</strong>  
تحكم كامل ورؤية شاملة للنظام.

<strong>النطاق:</strong>

<ul dir="rtl">
<li>جميع الفروع</li>
<li>جميع الطلبات</li>
<li>جميع الإعدادات</li>
<li>التكاملات</li>
<li>إدارة المستخدمين</li>
</ul>

---

### Admin (مدير فرع / مشرف تشغيل)

<strong>الوصف:</strong>  
مسؤول عن تشغيل فرع واحد فقط.

<strong>النطاق:</strong>

<ul dir="rtl">
<li>الفرع المخصص له</li>
<li>الطلبات الجارية</li>
<li>التحكم في الضغط ووقت التحضير</li>
</ul>

---

## ٤. الصفحة الرئيسية

### Super Admin يرى:

<ul dir="rtl">
<li>عدد الطلبات اليوم</li>
<li>الطلبات الجارية</li>
<li>متوسط وقت التحضير</li>
<li>حالة الاتصال بـ Foodics</li>
<li>الفروع المفتوحة / المغلقة</li>
<li>التنبيهات</li>
</ul>

### Admin يرى:

<ul dir="rtl">
<li>طلبات الفرع</li>
<li>الطلبات خلال ٣٠ دقيقة</li>
<li>حالة الفرع (نشط / موقوف)</li>
<li>مؤشر الضغط (منخفض / متوسط / مرتفع)</li>
</ul>

## ٦. قواعد النظام

<ol dir="rtl">
<li>لوحة التحكم لا تعدل المنيو أو الأسعار.</li>
<li>Foodics هو المصدر الأساسي للبيانات.</li>
<li>أي تعديل تشغيلي يجب أن يكون مؤقتًا وقابلًا للتتبع.</li>
<li>الأزرار الحساسة تحتاج تأكيد.</li>
<li>الصلاحيات تتحكم في ما يتم عرضه وتنفيذه.</li>
</ol>
