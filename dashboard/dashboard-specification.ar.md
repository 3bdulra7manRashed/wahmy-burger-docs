# مواصفات لوحة التحكم

---

## ١. نظرة عامة

لوحة التحكم هي نظام تشغيلي لإدارة طلبات الاستلام (Pickup) الأونلاين فقط.  
وليست بديلاً عن نظام نقاط البيع (Foodics).

### الأهداف
- متابعة الطلبات الأونلاين
- التحكم في تدفق الطلبات ومنع الزحام
- إدارة أوقات الاستلام (Pickup Slots)
- مراقبة التكامل مع Foodics
- توفير رؤية تشغيلية واضحة

---

## ٢. المستخدمون والصلاحيات

### Super Admin

**الوصف:**  
تحكم كامل ورؤية شاملة للنظام.

**النطاق:**
- جميع الفروع
- جميع الطلبات
- جميع الإعدادات
- التكاملات
- إدارة المستخدمين

---

### Admin (مدير فرع / مشرف تشغيل)

**الوصف:**  
مسؤول عن تشغيل فرع واحد فقط.

**النطاق:**
- الفرع المخصص له
- الطلبات الجارية
- التحكم في الضغط ووقت التحضير

## ٤. الصفحة الرئيسية

### Super Admin يرى:
- عدد الطلبات اليوم
- الطلبات الجارية
- متوسط وقت التحضير
- حالة الاتصال بـ Foodics
- الفروع المفتوحة / المغلقة
- التنبيهات

### Admin يرى:
- طلبات الفرع
- الطلبات خلال ٣٠ دقيقة
- حالة الفرع (نشط / موقوف)
- مؤشر الضغط (منخفض / متوسط / مرتفع)

---

## ٥. إدارة الطلبات

### البيانات المعروضة
- رقم الطلب
- الفرع
- وقت الاستلام
- الحالة
- طريقة الدفع
- كود الاستلام / QR

### صلاحيات Admin
- عرض طلبات الفرع
- عرض التفاصيل
- تغيير الحالة حسب السياسة
- إلغاء قبل بدء التحضير
- إضافة ملاحظة داخلية

غير مسموح:
- تعديل الأسعار
- تعديل محتوى الطلب

### صلاحيات Super Admin
- عرض جميع الطلبات
- فلترة متقدمة
- تصدير CSV

---

## ٦. التحكم في الـ Pickup Slots

### Super Admin
- تحديد مدة الـ Slot
- تحديد السعة الافتراضية
- إعدادات الطوارئ العامة

### Admin
- زيادة وقت التحضير مؤقتًا
- تقليل السعة مؤقتًا
- إيقاف استقبال الطلبات

أي تعديل مؤقت يجب أن يحتوي على:
- وقت بداية
- وقت نهاية
- سبب اختياري

---

## ٧. إعدادات الفروع

### Super Admin
- إضافة / تعديل فرع
- تحديد أوقات العمل
- وقت التحضير الأساسي
- تفعيل / تعطيل الفرع

### Admin
- عرض فقط
- تعديل وقت التحضير المؤقت

---

## ٨. المنيو (عرض فقط)

المنيو تتم مزامنتها من Foodics.

تتضمن:
- الأقسام
- المنتجات
- الأسعار
- الإضافات
- حالة التوفر

غير مسموح:
- إضافة
- تعديل
- حذف

الأزرار:
- Sync Menu
- عرض آخر مزامنة
- عرض أخطاء المزامنة

---

## ٩. التكامل مع Foodics

(Super Admin فقط)

يعرض:
- حالة الاتصال
- آخر Webhook
- الطلبات الفاشلة
- محاولات إعادة الإرسال
- السجلات

---

## ١٠. التقارير

### Super Admin
- عدد الطلبات يوميًا
- ساعات الذروة
- متوسط وقت التحضير
- استخدام الـ Slots

### Admin
- تقارير الفرع فقط
- عرض فقط

---

## ١١. قواعد النظام

١. لوحة التحكم لا تعدل المنيو أو الأسعار.  
٢. Foodics هو المصدر الأساسي للبيانات.  
٣. أي تعديل تشغيلي يجب أن يكون مؤقتًا وقابلًا للتتبع.  
٤. الأزرار الحساسة تحتاج تأكيد.  
٥. الصلاحيات تتحكم في ما يتم عرضه وتنفيذه.

---

## ١٢. حالات الخطأ

- انقطاع Foodics → تنبيه واضح  
- امتلاء Slots → منع الطلب  
- فشل Webhook → إعادة محاولة + تسجيل  
- تجاوز وقت التحضير → تحذير  

---

## الخلاصة

- لوحة تحكم موحدة  
- أدوار واضحة  
- تحكم تشغيلي فقط  
- المنيو تُدار من Foodics  
- مصممة للتشغيل الواقعي
